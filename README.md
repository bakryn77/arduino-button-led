# Arduino Button–LED Project  

A simple beginner Arduino project that turns an LED on/off using a pushbutton.  
 
---

##  Components Used
- Arduino Uno (or any compatible board)  
- 1x LED  
- 1x 220Ω–330Ω resistor  
- 1x Pushbutton  
- Breadboard  
- Jumper wires  

---

## Overview
When the pushbutton is pressed, the LED turns on.  
When released, the LED turns off.

This uses:
- digital input (reading a button)  
- digital output (controlling an LED)  
- basic wiring
